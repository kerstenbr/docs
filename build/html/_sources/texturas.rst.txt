************************
Bugs gráficos e texturas
************************

[h1]Caso você esteja experienciando bugs, erros e texturas rosa e preto, você deverá seguir o tutorial abaixo para consertar o problema.[/h1]

[b]Para manter uma melhor didática, este tutorial estará disponível aqui por texto em etapas e, também, por vídeo. Veja abaixo:[/b]

[b][url=https://drive.google.com/drive/folders/11uRktjIcb2r9QnFcAY0Qo_7RQ-yHz6yR?usp=sharing]Link para download dos arquivos necessários[/url]. Recomendamos este por motivos de segurança e praticidade; sem propagandas ou arquivos estranhos/possivelmente maliciosos.[/b]

[h1][url=https://www.youtube.com/watch?v=hge8ZWhOvcA]Tutorial em vídeo[/url][/h1] [b]O vídeo não é de nossa autoria. Todos os créditos vão para o devido autor do mesmo.[/b]

[b]Guia prático em etapas:[/b]
[list]
[*]Baixe o arquivo compactado acima;
[*]copie o arquivo baixado para a pasta [i]addons[/i] do seu Garry's Mod;
[*]extraia o arquivo na pasta addons e garanta que a estrutura fique assim:
```
🗁 garrysmod/addons/css_content_addon/
⮡ 🗁 materials/
⮡ 🗁 models/
⮡ 🗁 particles/
⮡ 🗁 sound/
```
.. admonition: warning
ATENÇÃO, VERIFIQUE E GARANTA QUE A ESTRUTURA NÃO FIQUE ASSIM:
```
🗁 garrysmod/addons/css_content_addon/css_content_addon/
```

[*][b]por fim, reinicie o jogo.[/b]
[/list]

[h1]Pronto! Isso deve servir para consertar todos os erros.[/h1]
